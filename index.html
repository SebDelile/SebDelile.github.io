Réalisations

  • Kasa

    Application d'annonces de location immobilière avec une galerie et des pages pour chaque bien. Application réalisée en React avec Create-react-app et React-router, entièrement testée avec Jest, Testing-library et Cypress. Mise en place d'une CLI avec GitHub action et d'un déploiement sur Netlify

  • SportSee

    Page de profil d’une application sportive contenant un dashboard analytique avec plusieurs graphiques et dont les données sont récupérées sur une API. Application réalisée en React avec Create-react-app et React-router. Les graphiques sont générés en svg avec D3.js et les appels à l'API sont faits avec Axios. Docker et Docker-compose ont été mis en place afin de déployer des images de l'application (l'API n'est disponible qu'en local).

  • MarbleBowl

    Projet communautaire de clicker game où l’utilisateur découvre les nouvelles fonctionnalités et feuilles de style à mesure de sa progression dans la narration. Application réalisée en React avec Create-react-app. Etat global de l'application pris en charge par Redux avec redux-toolkits Développement en méthodologie agile SCRUM avec des sprints de 3 semaines, kanban, code-review et planning poker.

  • React datatable

    Plugin React de table de données offrant des fonctionnalités de pagination, de tri, de filtre et 2 méthodes de responsive (scroll horizontal ou info supplémentaires au clic). Le plugin est créé avec Create-react-app et n'utilise qu'une dépendance supplémentaire : @Emotion/css pour gérer la personnalisation de style en CSS-in-JS. Il est entièrement couvert par des tests unitaires et CLI mise en place avec Github action.

  • Portfolio

    Page web personnelle proposant un aperçu de mes préférences, de mes projets et de mon parcours. Page réalisée en React et compilée avec Gatsby pour générer une page statique optimisée. Utilisation de nombreux plugin Gatsby pour gérer les imports JSON, les images, les métadonnées, ...

  • Les petits plats

    Preuve de concept d'une page web de recette avec un moteur de recherche intégré par tags et/ou champ libre. Production de 2 algorithmes pour la recherche avec les schémas logiques et comparaison de performances. Page intégrée avec Bootstrap 4 et vanilla JS, utilisation de Sass pour personnaliser le thème bootstrap
